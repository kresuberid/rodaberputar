## ğŸ“‹ Tentang Aplikasi

**Roda Berputar** (Spin Wheel Picker) adalah aplikasi web berbasis React yang dirancang untuk membantu pengguna melakukan pengacakan nama, item, atau pemenang secara adil, transparan, dan menyenangkan. Aplikasi ini dibangun dengan teknologi modern untuk memastikan performa yang cepat dan tampilan UI/UX yang premium.

Dikembangkan oleh **Febri Suryanto** dengan fokus pada estetika 3D Glassmorphism dan optimasi SEO/GEO (Generative Engine Optimization).

## âœ¨ Fitur Utama

*   **Antarmuka Modern (3D Glassmorphism):** Tampilan visual yang menawan dengan efek kaca dan pencahayaan dinamis.
*   **100% Responsif:** Bekerja sempurna di Desktop, Tablet, dan Smartphone.
*   **Tanpa Login:** Langsung pakai tanpa perlu mendaftar akun.
*   **6 Tema Premium:** Termasuk Vegas (Merah), Fortune (Teal), Royal (Ungu), Midnight (Hitam), Ocean (Biru), dan Rainbow.
*   **Dukungan Multi-Bahasa:** Tersedia dalam 12 bahasa global (Indonesia, Inggris, Spanyol, Arab, Jepang, Korea, dll).
*   **Efek Suara Realistis:** Suara "ticking" saat berputar dan fanfar kemenangan yang meriah.
*   **Konfigurasi Lanjutan:** Atur durasi putaran, hapus pemenang otomatis, dan manajemen daftar nama.
*   **SEO & GEO Optimized:** Terstruktur agar mudah ditemukan oleh mesin pencari dan AI.

## ğŸš€ Instalasi & Menjalankan (GitHub)

Ikuti langkah-langkah berikut untuk menjalankan project ini di komputer lokal Anda:

### Prasyarat
Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/) (versi 16 atau lebih baru).

### Langkah 1: Clone Repository
Buka terminal dan jalankan perintah berikut:

```bash
git clone https://github.com/username/roda-berputar.git
cd roda-berputar
```

### Langkah 2: Instalasi Dependensi
Instal paket yang diperlukan menggunakan npm atau yarn:

```bash
npm install
# atau
yarn install
```

### Langkah 3: Menjalankan Development Server
Jalankan aplikasi dalam mode pengembangan:

```bash
npm start
# atau
yarn start
```

Buka browser dan akses `http://localhost:3000`.
